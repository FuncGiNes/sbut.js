# SBUT Engine

SBUT Engine is a lightweight JavaScript game engine.

## CDN

```html
<script src="https://cdn.jsdelivr.net/gh/FuncGiNes/sbut.js@v1.0.0/sbut.js"></script>
```

## Features

- 2D Game Engine
- Physics
- Camera
- Audio
- Scene System
- Touch Controls
- Save System
